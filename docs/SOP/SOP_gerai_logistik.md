# SOP Gerai Logistik

## Tujuan
Menjamin distribusi dan rantai pasok koperasi berjalan lancar.

## Prosedur
1. Perencanaan distribusi hasil produksi
2. Pengadaan sarana transportasi dan gudang
3. Pengiriman barang ke gerai atau anggota
4. Monitoring perjalanan & stok barang
5. Laporan logistik bulanan
