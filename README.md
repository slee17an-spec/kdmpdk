# Koperasi Desa Merah Putih (KDMP) - Desa Kombo

Repositori ini berisi dokumentasi, struktur organisasi, dan informasi operasional Koperasi Desa Merah Putih Desa Kombo.  
Tujuan repositori:
- Dokumentasi internal koperasi
- Portal digital untuk anggota
- Basis pengembangan website dan dashboard koperasi

## Struktur Portal
Portal utama berisi:
1. Kantor Koperasi
2. Gerai Pangan Mandiri Desa
   - Sub Gerai Produksi Tanaman
   - Sub Gerai Peternakan Terpadu
   - Sub Gerai Perikanan Terpadu
   - Sub Gerai Pakan Mandiri
   - Sub Gerai Pupuk & Pengolahan Limbah
3. Gerai Sembako
4. Gerai Simpan Pinjam
5. Gerai Klinik Kesehatan Desa
6. Gerai Apotik
7. Gerai Cold Storage
8. Gerai Logistik

Setiap gerai memiliki dokumen terpisah yang menjelaskan unit usaha dan SOP.
